K-Means¶

In this notebook we will look at two examples. The first example is the classic iris dataset. We will first apply K-Means to cluster the iris dataset, then plot on a 2-dimensional graph. The second example is a dataset of 12 variables. In both examples we will use principal component analysis to reduce to 2-dimensions to plot on a 2-dimentional graph.

Learning objectives:

  • K-Means
  • Hyperparameter experimenting
  • Dimension reduction

Two examples:

  • dataset 1: iris
  • dataset 2: wine from library(rattle.data)
